# Films-d-horreur

README — Projet Web : The Horror Vault
1. Comprendre le projet
1.1 Quel est le but principal du site ?

Le but principal du site est d’informer et divertir les amateurs de films d’horreur en leur proposant une sélection de films avec un résumé, des informations et une navigation simple.

L’utilisateur peut découvrir de nouveaux films d’horreur, lire un résumé sans spoiler et explorer différents films selon ses préférences.

L’action principale attendue de l’utilisateur est consulter la liste des films et cliquer sur un film pour en découvrir plus.

Le site est conçu pour durer dans le temps, car il peut être enrichi avec de nouveaux films.

1.2 Qui sont les utilisateurs ?

Le site s’adresse principalement aux fans de films d’horreur et aux curieux qui souhaitent découvrir ce genre de cinéma.

Les utilisateurs peuvent être :

étudiants

amateurs de cinéma

personnes cherchant un film à regarder

Habitudes numériques

Les utilisateurs consultent souvent ce type de site depuis leur smartphone ou leur ordinateur, généralement pour trouver rapidement un film intéressant.

Contexte d’utilisation

usage personnel

navigation rapide

recherche d’informations simples sur un film

Niveau de confort avec le web

La majorité des utilisateurs ont un niveau intermédiaire avec le web et sont habitués à naviguer sur des sites de films ou de streaming.

1.4 Identité du projet
Valeurs du site

passion pour le cinéma d’horreur

découverte de films

simplicité de navigation

partage de culture cinématographique

Émotions à transmettre

Le site doit transmettre :

mystère

tension

curiosité

plaisir de découvrir des films

Ambiance visuelle

L’ambiance visuelle du site doit refléter l’univers du cinéma d’horreur :

couleurs sombres (noir, bleu, jeune, blanc)

contrastes forts

images d’affiches de films

interface simple et immersive

Le design doit rester lisible et facile à naviguer, même avec une atmosphère sombre.
